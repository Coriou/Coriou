## Open source

[up-redis](https://github.com/Coriou/up-redis) -- Self-hosted Upstash Redis alternative. Drop-in compatible REST proxy for any Redis 6+. 232 tests, weekly SDK compatibility CI. TypeScript / Bun / Hono.

[up-vector](https://github.com/Coriou/up-vector) -- Self-hosted Upstash Vector alternative. Vector search backed by Redis Stack. Full metadata filter parser. 198 tests. TypeScript / Bun / Hono.

[WaveKit](https://github.com/Coriou/WaveKit) -- Real-time SDR framework decoding 8 radio protocols simultaneously (aircraft, ships, pagers, digital voice) from a single receiver. 30K LoC monorepo. TypeScript / Node.js / Fastify.

[py-audio-box](https://github.com/Coriou/py-audio-box) -- Dockerized voice cloning & TTS toolkit. Clone any voice from a URL or file, synthesize in 10 languages. Built on Qwen3-TTS. Python / Docker.
